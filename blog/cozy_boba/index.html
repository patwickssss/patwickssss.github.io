Project overview

Cozy Boba is a full-stack café management application built with HTML, CSS, JavaScript, PHP, and MySQL. It provides a structured platform for managing menu items, processing customer orders, and tracking sales records through a database-driven workflow.

The system supports milk tea and café items such as pizzas, nachos, and donuts. It was designed to reduce manual work, improve ordering efficiency, and serve as a foundation for future enhancements like inventory control and analytics.

Key features

Cozy Boba combines user-friendly front-end design with database-driven back-end functionality to help small café operators manage daily sales and customer orders smoothly.

  • Menu management for milk tea, pizzas, nachos, and donuts.
  • Order processing with dynamic pricing and transaction recording.
  • Sales tracking with MySQL-backed storage and retrieval.
  • Responsive front-end interface for desktop and mobile use.
